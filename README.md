#  ts-vue-hello
This is a helloworld demo base on Typescript with vue+vue-router+vuex(vuex-class)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
